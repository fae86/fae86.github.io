Guía de uso del addon "My Wizard FAE86"
Instaladores de Build
El addon permite instalar builds en varias versiones de dispositivos. Aquí se detallan las opciones disponibles:

PC: Instala la versión del build compatible con computadoras.
Smart TV: Versión diseñada para televisores inteligentes.
Smartphone: Build adaptado para dispositivos móviles.
Tablet: Versión de build optimizada para tabletas.
El addon detectará automáticamente el dispositivo y ofrecerá las versiones más adecuadas para cada uno.

Apartado de Mantenimiento
Dentro de la sección de mantenimiento, encontrarás varias herramientas útiles para optimizar el rendimiento de Kodi. Aquí están las principales funciones:

Limpiador de Paquetes:

Elimina paquetes de addons o actualizaciones antiguas que ya no se utilizan, ayudando a liberar espacio y mejorar el rendimiento de Kodi.
Limpiador de Miniaturas:

Elimina miniaturas de videos, imágenes o contenido innecesario para optimizar la base de datos y mejorar la velocidad de carga de Kodi.
Fresh Start (Restablecer configuración):

Esta función restablece Kodi a su configuración predeterminada de fábrica, eliminando todos los datos y configuraciones personalizadas. Ideal para una instalación limpia o cuando se experimentan problemas.
Ajustes Avanzados:

Acceso a configuraciones más profundas para personalizar Kodi y su comportamiento de manera avanzada.
Editor de Whitelist:

Permite gestionar una lista blanca de fuentes o servidores que están permitidos para acceder a Kodi, mejorando la seguridad.
Restaurador de Configuraciones de Gui/Skin:

Restaura la interfaz de usuario (GUI) o el skin a su configuración predeterminada o a una versión guardada anteriormente, facilitando la personalización visual de Kodi.
Forzar Cerrar Kodi:

Si Kodi se congela o no responde, esta opción fuerza su cierre de manera segura.
Speedtest:

Mide la velocidad de tu conexión a internet para asegurarte de que Kodi pueda funcionar de manera óptima, especialmente cuando se transmiten videos o contenido de alta calidad.
Visor del Log de Kodi:

Permite visualizar los registros (logs) de Kodi, lo que ayuda a diagnosticar errores o problemas técnicos en el sistema.
Funciones Adicionales
El addon también incluye otras herramientas útiles que mejoran la experiencia del usuario y la administración de Kodi. La interfaz es sencilla e intuitiva, lo que facilita su uso incluso para usuarios principiantes.
